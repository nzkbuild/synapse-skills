# Recall Sessions

1. Search past sessions by keyword:
```bash
synapse --echo "your search query"
```

2. This searches through:
   - Routing diary files (date-based)
   - Session memory (recent sessions)
   - Archived diary entries

3. Results show:
   - **Date/time** of the routing session
   - **Task** that was routed
   - **Skills** that were selected

4. Use this to:
   - Find what skills were used for similar past tasks
   - Review routing patterns over time
   - Identify skills that keep appearing for your work
